Lexikon

28 hours ago

Besoins du générateur de dictionnaire:

  • décrire un alphabet et l'ordre lexicographique associé. Seules les lettres de l'alphabet seront reconnues pour constituer les mots du dictionnaire
  • les langues comportant des mots composés doivent pouvoir définir une liste de séparateurs (comme le - en français)
  • un mot est dans une successions de lettres et de séparateurs
  • décrire la liste exhaustive des catégories auxquelles les mots appartiennent
  • décrire la list exhaustive des traits possibles pour les mots du dictionnaire
  • décrire des mots
  • chaque mot construit est décoré par une et une seule catégorie et une liste (potentiellement vide) de traits
    • un mot peut apparaitre plusieurs fois dans le dictionnaire avec des catégories et des traits différentes. Par exemple en français le mot français est à la fois un adjectif et un nom commun
  • amender des radicaux en ajoutant des préfixes ou suffixes via des règles de réécriture
    • il est possible que les radicaux soient rognés aux extrémités. En français par exemple le pluriel de cheval est chevaux, ce qui peut être décrit comme la transformation du radical cheval en ôtant le l final et en ajoutant ux. Cette transformation s'applique à touts les noms qui se terminent par al à l'exception de quelques-uns. Par exemple festival.
    • il doit être possible d'établir des priorités dans les règles de réécriture
    • une seule règle doit être appliquée à la fois.
    • une règle de réécriture ne peux pas changer la catégorie d'un mot
    • une règle de réécriture se base sur la forme du mot, mais doit aussi pouvoir utiliser la catégorie et les traits du mot réécrit.

Pour couvrir ces besoins, nous définissons un langage de description de dictionnaire nommé Diko.

Diko

24 hours ago

Le langage Diko permet de décrire des dictionnaires. Afin d'être le plus interopérable possible, peu de concepts existent nativement, et quasi tous doivent être définis. Un dictionnaire consiste en plusieurs parties décrites dans cette section.

  1. Spécification de la locale
  2. L'alphabet
  3. Les séparateurs
  4. Les catégories
  5. Les traits
  6. Le point d'entrée
  7. Les lexiques
  8. Annexes

Les traits

2 hours ago

Syntaxe

"tags" { (long-name "as" name ";" |: "group" long-name "{" { long-name "as" name ";" } "}") }

Description

La section tags permet de définir les traits d'un mot.

Un trait peut être défini seul ou dan un groupe. Les traits définis dans un même groupe sont mutuellement exclusifs, les autres peuvent cohabiter pour une même entrée.
