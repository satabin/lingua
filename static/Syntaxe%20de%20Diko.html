Syntaxe de Diko

8 hours ago

Toutes les règles de syntaxe du langage Diko sont données ci-dessous au format railroad.

AlphabetRailroad

"alphabet" { char } ";"

CategoriesRailroad

"categories" { long-name "as" name ";" }

DefaultRailroad

[: "@" name ] { ("+" name |: "-" name) }

EntryPointRailroad

"main" "=" long-name ";"

EntryRailroad

(word |: rewrite-rule)

LexikonRailroad

"lexikon" long-name default "{" { entry } "}"

PatternRailroad

[: ">" ] { ("." | char |: "\" number)} [: "<" ] [: "/" default ]

ReplacementRailroad

[: ">" ] { (char |: "\" number) } [: "<" ] [: "/" { ("+" name |: "-" name) } ]

RewriteRuleRailroad

"rewrite" long-name default "{" { "rule" pattern "=>" replacement ";" } "}"

SeparatorsRailroad

"separators" { char } ";"

TagsRailroad

"tags" { (long-name "as" name ";" |: "group" long-name "{" { long-name "as" name ";" } "}") }

WordRailroad

[{: char }] [: "/" default ] ";"
