L'alphabet

3 hours ago

Syntaxe

"alphabet" { char } ";"

Description

La section alphabetpermet de spécifier les lettres utilisées dans la langue pour laquelle le dictionnaire est écrit. Chaque lettre est définie comme étant un caractère non vide et elles sont séparées par un nombre arbitraire (voire nul) d'espaces. L'ordre de déclaration des lettres dans l'alphabet définit l'ordre lexicographique.

L'alphabet contient les lettres et pas les séparateurs (tel que le - en français) qui sont déclarés dans leur propre section. Ainsi les séparateurs ne participent pas à l'ordre lexicographique.
